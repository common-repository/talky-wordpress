This plugin adds sound events for WordPress

== Description ==

This plugin adds sound events for publishing and deleting posts or pages, plugin activation and deactivation and theme activation. We're also thinking about adding an event for theme deactivation, please let us know what you think about it.

== Installation ==

1. Download the `talky-wordpress.zip` file.
2. Unzip the `talky-wordpress.zip` file.
3. Upload the `talky-wordpress` directory to your plugins directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Go to the 'Settings' administration menu, you shoud see a 'Talky WordPress' menu item.
6. Choose the volume and sound theme options.
7. Click the 'Save Options' button.

== Frequently Asked Questions ==

= Where is the 'Talky WordPress' administration menu? =

It's under 'Settings'.

= Are you sure that your plugin does not work for WordPress versions older than 2.7? =

No, we are not sure. It might work. We just haven't tested it yet for older versions.

= Does your plugin work for WordPress Multi-User? =

Yes.
